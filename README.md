##Selenium Advanced Repository
Prerequistes:<br/>
Java 11 or More<br/>
Check Java Version - <br/>
java --version <br/>
Apache Maven - <br/>
set env variable MAVEN_HOME and edit Path Variable till 'bin' path <br/>

####POM.xml 
add required dependencies<br/>
1. testng<br/>
2. maven-surefire-plugin<br/>
3. mvn plugin<br/>

####Run testcases <br/>
mvn clean test
